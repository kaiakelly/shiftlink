## 驗收步驟（MVP）

### 0) 安裝與啟動

```bash
cd shiftlink
git pull
npm install
npx expo start -c
```

### 1) 登入／註冊

- admin: `admin` / `112211`
- user1: `user1` / `112211`（VIP）
- user2: `user2` / `112211`

### 2) 日曆

- 點班次按鈕 → 自動跳下一日
- VIP 點 emoji → 自動跳下一日
- 🗑 → 清班（班別 + emoji），並跳下一日
- 「儲存 / 已儲存」狀態切換

### 3) Feed

- 廣場／換班牆發帖（技能不必選）
- Swap 只顯示同公司帖子
- 帖子作者行可進入私訊（遊客會導去登入）

### 4) 私訊＋換更

- 私訊列表可見 thread；點入可文字對話
- 相簿／相機可發圖（列表預覽顯示 [圖片]）
- 換更：發起提議卡 → 對方同意/拒絕
- 同意後可選擇「更新日曆」，更新成功會顯示 confetti 並寫入系統訊息

### 5) Admin（只限 admin）

- 我的 → Admin
- App 設定：VIP 價格 / 公告 / 禁詞可改
- CRUD：公司 / 職位 / 技能 / 班次類型 / 用戶（VIP/禁言/角色）
